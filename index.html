Deleting binary and object files from a compiled solution in Visual Studio

find . -iname "bin" -o -iname "obj" | xargs rm -rf

find - get all files and directories which match the criteria specified. ". " - all => find all. -iname - case insensitive. "bin" - look for files and directories called "bin". -o - or "obj" - look for files and directories called "obj". | - pipe operator. pass the output of the previous command to the following command. In this case, pass all files and directories named "bin" and "obj" to the rm function. rm - remove -rf - do the remove recursively, and force it.
